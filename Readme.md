# AI Rule-Based Chatbot

## Description
This is a simple AI chatbot developed using Python. It responds to user inputs based on predefined rules using if-else statements.

## Features
- Greets the user
- Tells the current date and time
- Tells jokes
- Answers basic questions
- Exits when the user types "bye"

## Technologies Used
- Python 3

## How to Run
1. Open the project folder.
2. Run:
   python chatbot.py
